# Despliegue de Microservicios con Docker

Este proyecto tiene como objetivo el despliegue de varios microservicios utilizando Docker.

## Requisitos

1. Tener instalado y activo [Docker Desktop](https://www.docker.com/products/docker-desktop) en tu m√°quina.
2. Tener `git` instalado para clonar el repositorio.
3. Un cliente para hacer pruebas de los endpoints, como [Postman](https://www.postman.com/).

## Pasos para el despliegue

### 1. Clonar el repositorio

```bash
git clone https://github.com/fernandoEncalada/MicroservicesTechnical.git
```
### 2. Ejecutar el siguiente comando

```bash
docker compose up -d
```
